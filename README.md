# 🛒 E-Commerce SQL Analysis

## 📌 About the Project
Analysis of a UK-based e-commerce company's sales data from 2009 to 2011 using SQL and Python.

## 📊 Dataset
- **Source:** UCI Online Retail II (Kaggle)
- **Size:** 1,067,371 rows, 8 columns
- **Period:** 2009-2011

## 🛠️ Technologies Used
- Python, Pandas, NumPy
- SQLite, SQL
- Matplotlib, Seaborn
- Jupyter Notebook

## 🔍 Analyses
- Top 10 customers by revenue
- Monthly revenue trend
- Sales analysis by country
- Best-selling products
- RFM Customer Segmentation

## 💡 Key Findings
- **43% of customers are lost** → A win-back campaign is recommended
- Sales peak every **October-November** → Christmas season effect
- **EIRE (Ireland)** generates the 2nd highest revenue with only 5 customers → likely B2B
- **WHITE HANGING HEART** is both the best-selling and most profitable product
